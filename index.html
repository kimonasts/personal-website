Projects

Educational Quiz Platform

This project is an Educational Quiz Platform designed for my IB Computer Science Internal Assessment. It utilizes a Java Swing framework with a WindowBuilder plugin to create a user-friendly interface. The system caters to two types of users: Admin and Student. The Admin role involves creating a collection of quizzes, each containing questions with four possible answers. The Admin can manage the quizzes, add new questions, and customize the content to suit different subjects or topics. On the other hand, the Student role is focused on taking the quizzes. Students can access the available quizzes, answer the questions, and submit their responses. Once the quiz is completed, the system provides immediate feedback by displaying the final score and the amount of time taken to complete the quiz.
